# AI Collective Document System (Independent)

Token source: `tokens/brand-tokens.json`

## Default page system
- Page size: Letter
- Body role: `body`
- Base line spacing: 1.35

## Heading map
- Document title -> `h1`
- Major section heading -> `h2`
- Subsection heading -> `h3`
- Tertiary heading -> `h4`
- Caption and footnote -> `caption`

## Paragraph rules
- Target 60 to 80 characters per line.
- Keep paragraphs left-aligned.
- Use one idea per paragraph.

## List rules
- Keep parallel grammar across bullets.
- Target up to seven items per list.
- Use concise action-forward phrasing.

## Callout rules
- Use accent surfaces, not orange fills.
- Set callout heading style to `label`.
- Limit callout width to 80% of text column.

## Voice rules
- Keep language community-first and practical.
- Use approved voice anchors naturally, not repetitively.
